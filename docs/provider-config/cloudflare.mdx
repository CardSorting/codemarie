### Getting Started with Cloudflare Workers AI

To use Cloudflare Workers AI with Cline, you'll need a Cloudflare account, your Account ID, and an API Token.

1.  **Find your Account ID:**
    *   Log in to the [Cloudflare dashboard](https://dash.cloudflare.com/).
    *   Your Account ID is visible in the URL of your dashboard (e.g., `dash.cloudflare.com/<your-account-id>`).
    *   Alternatively, you can find it on the **Overview** page of any of your domains.

2.  **Create an API Token:**
    *   Navigate to **My Profile > API Tokens** or go to [dash.cloudflare.com/profile/api-tokens](https://dash.cloudflare.com/profile/api-tokens).
    *   Click **Create Token**.
    *   Use the **Workers AI (Read)** template or create a custom token with **Account > Workers AI > Edit** permissions.
    *   Copy and store your API token securely.

### Supported Models

Cline supports the following Cloudflare Workers AI models:

-   `@cf/zai-org/glm-4.7-flash` (Recommended) - Fast and efficient multilingual text generation with a 131,072 token context window.
-   `@cf/meta/llama-3.3-70b-instruct-fp8-v2` - Powerful, high-performance model for complex reasoning and general-purpose tasks.
-   `@cf/deepseek-ai/deepseek-r1-distill-qwen-32b` - DeepSeek R1 Distill Qwen 32B model, optimized for reasoning and efficiency.

For a full list of available models and their capabilities, refer to the [Cloudflare Workers AI Models documentation](https://developers.cloudflare.com/workers-ai/models/).

### Configuration in Cline

1.  **Open Cline Settings:** Click the settings icon (⚙️) in the Cline panel.
2.  **Select Provider:** Choose **Cloudflare Workers AI** from the **API Provider** dropdown.
3.  **Enter Account ID:** Paste your Cloudflare Account ID into the **Cloudflare Account ID** field.
4.  **Enter API Token:** Paste your Cloudflare API token into the **Cloudflare API Token** field.
5.  **Select Model:** Choose your desired model from the **Model** dropdown.

### Pricing and Limits

Cloudflare Workers AI pricing is based on the model and usage.
-   **GLM-4.7-Flash:** Approximately $0.06 per 1M input tokens and $0.40 per 1M output tokens.
-   **Free Tier:** Cloudflare offers a free tier for Workers AI with daily limits.

Refer to the [Cloudflare Workers AI Pricing](https://developers.cloudflare.com/workers-ai/platform/pricing/) for the most up-to-date information.
